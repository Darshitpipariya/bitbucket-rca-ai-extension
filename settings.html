Bitbucket API Token (For Private Repositories):

To access private repositories, you need to create a Bitbucket API token:

  1. Go to Bitbucket ‚Üí Personal Settings ‚Üí App passwords
  2. Click "Create app password"
  3. Give it a label (e.g., "RCA Helper")
  4. Select permissions: Repositories: Read
  5. Click "Create" and copy the generated token

Note: You'll also need your Atlassian account email (found under Email Aliases in Personal settings).
